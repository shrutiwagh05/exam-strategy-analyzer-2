# 📊 Exam Strategy Analyzer

## 🚀 Overview

Exam Strategy Analyzer is a data-driven dashboard that classifies student exam attempt behavior into **Fast**, **Balanced**, and **Slow** strategies based on time distribution across questions.


## 🎯 Problem Statement

Analyze exam attempt logs (timestamps per question) and classify student strategies using measurable timing features without subjective interpretation.


## 🧠 Approach

The classification is based on:

* **Average Time per Question**
* **Variation in Response Time**

**Strategy Logic:**

* Fast Attempt → Lower average time
* Balanced Attempt → Moderate average time
* Slow Attempt → Higher average time

All decisions are **objective and data-driven**.

---

## 📊 Features

* Strategy classification (Fast / Balanced / Slow)
* Time-based analysis
* Interactive dashboard with visual insights
